How to properly set up an PHP file

To start making a PHP, you should open the terminal window and type in the sudo nano /etc/apache2/http.conf and press

Following this, press the Ctrl+ W which will bring up a search bar

Next you should search for php and hit the enter key

After this you will see a screen filled with nano Sudo codes

While on this screen you should delete the # from the text marked #LoadModule php7_module libphp7.co, the text in the seventh bar.

Following this you should hit the Ctrl+ O alomg with an enter to save this change

Next to exit nano, you should press Crtrl and the x key

Type sudo apachectl restart and press enter to start the next phase
